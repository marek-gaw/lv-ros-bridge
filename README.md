# lv-ros-bridge

###Required: 
* LabVIEW 2013
* lib_websockets by Intelligent Energy (Websockets for LabVIEW)
* ROS Kinetic
* package: rosbridge-suite

This project provides simple parser of ROS messages for LabVIEW.
